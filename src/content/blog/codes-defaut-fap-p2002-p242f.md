> **En bref —** Les **codes défaut FAP** (P2002, P242F, P2452, P244A…) signalent un problème de filtre à particules : **efficacité sous le seuil**, **cendres accumulées**, **capteur de pression** ou **régénération** défaillante. Un code ne veut **pas** forcément dire « FAP à changer » : tant que la céramique est saine, un **nettoyage** avec effacement des défauts règle souvent le problème. La clé : un **diagnostic** pour interpréter le code et traiter la vraie cause, pas seulement l'effacer.

Votre valise de diagnostic affiche un P2002 ou un P242F et vous ne savez pas quoi en penser ? Ces codes ne sont pas une condamnation. Voici ce qu'ils signifient et, surtout, ce qu'il faut faire — au-delà du simple effacement.

## Que sont les codes défaut FAP ?

Les **codes défaut** (DTC) sont des messages enregistrés par le calculateur quand il détecte une anomalie. Les codes liés au **FAP** concernent le filtre à particules, ses capteurs ou ses régénérations. Ils commencent souvent par **« P2 »** ou **« P244 »**. Ils orientent le diagnostic, mais ne disent pas à eux seuls quelle réparation effectuer.

## Comment lit-on un code défaut FAP ?

On lit les codes avec un **outil de diagnostic** (valise OBD) branché sur la prise du véhicule. L'outil affiche le code et son libellé. Mais lire un code ne suffit pas : il faut l'**interpréter** au regard de l'état réel du filtre (contre-pression, suies, cendres). C'est là qu'intervient le savoir-faire d'un professionnel.

## P2002 : efficacité du FAP sous le seuil

Le code **P2002** signifie « efficacité du filtre à particules **sous le seuil** ». En clair, le FAP ne filtre plus aussi bien qu'attendu, généralement parce qu'il est **colmaté** ou que ses capteurs détectent une anomalie. C'est l'un des codes FAP les plus courants. Bonne nouvelle : sur un filtre sain mais encrassé, un **nettoyage** résout souvent le problème.

## P242F : accumulation de cendres dans le FAP

Le code **P242F** indique une **restriction du FAP par accumulation de cendres**. Contrairement aux suies (qui se brûlent à la régénération), les **cendres** ne se régénèrent pas et s'accumulent durablement. Un **nettoyage en profondeur** (avec dépose) permet d'évacuer une grande partie de ces cendres — ce qu'une simple régénération ne fait pas.

## P2452 / P2453 : capteur de pression du FAP

Les codes **P2452 / P2453** concernent le **capteur de pression** du FAP (circuit ou plage). Ils peuvent signaler un capteur défaillant ou un problème de mesure, plutôt qu'un filtre forcément colmaté. D'où l'importance du diagnostic : remplacer le filtre alors que c'est le **capteur** qui est en cause serait une dépense inutile.

## P244A / P244B : pression différentielle anormale

Les codes **P244A** (pression différentielle trop basse) et **P244B** (trop élevée) renseignent sur l'état du filtre via la **différence de pression** entre son entrée et sa sortie. Une pression trop élevée évoque un **colmatage** ; une pression anormalement basse peut indiquer un capteur ou un filtre défectueux. Là encore, l'interprétation prime.

## P2458 / P2459 : durée ou fréquence de régénération

Les codes **P2458 / P2459** concernent la **régénération** : durée anormale ou fréquence inadaptée. Ils traduisent souvent un FAP qui **n'arrive plus à se régénérer correctement** — typiquement sur un véhicule à trajets courts. Le traitement passe par le nettoyage et, surtout, par la correction de la **cause** (usage, EGR, additif).

## Les codes additif (PSA) : P1471 et apparentés

Sur certains véhicules (notamment PSA à FAP additivé), des codes spécifiques signalent un problème lié à l'**additif** (type Eolys) qui facilite la régénération. Un **réservoir d'additif** vide ou un défaut du système peut déclencher ces codes et favoriser le colmatage. Vérifier l'additif fait partie du diagnostic sur ces modèles.

## Un code FAP signifie-t-il un FAP à changer ?

**Non**, pas nécessairement — et c'est le point le plus important. Beaucoup de codes FAP traduisent un **colmatage** (nettoyable) ou un **capteur** défaillant, pas un filtre détruit. Tant que la céramique est saine, le **[nettoyage FAP](/nettoyage-fap)** règle souvent la situation. Le remplacement est réservé aux filtres réellement irrécupérables.

## Suies ou cendres : deux problèmes différents

Les **suies** se brûlent à la régénération ; les **cendres** (issues notamment de l'huile) s'accumulent définitivement. Un code comme P242F pointe les cendres, qu'un nettoyage en profondeur peut évacuer. Distinguer suies et cendres aide à comprendre pourquoi une simple régénération forcée ne suffit pas toujours — et pourquoi le nettoyage avec dépose est plus complet.

## Que faire après lecture d'un code FAP ?

La bonne démarche : **ne pas se contenter d'effacer**, mais réaliser un **diagnostic complet** (codes + contre-pression + état de l'additif et de l'EGR). On identifie ainsi la vraie cause, puis on agit : nettoyage, remplacement de capteur, traitement de l'EGR… Effacer sans comprendre, c'est s'exposer au retour du code.

## Peut-on simplement effacer le code ?

Avec un outil OBD, on peut effacer un code — mais **effacer n'est pas réparer**. Si la cause demeure (FAP colmaté, capteur HS), le code **reviendra**, souvent rapidement. Pire, rouler en ignorant un FAP colmaté aggrave la situation. L'effacement n'a de sens qu'**après** avoir traité la cause réelle.

## Quand un nettoyage FAP résout le code

Quand le code traduit un **colmatage** sur un filtre sain (suies, cendres), le nettoyage en profondeur, suivi de l'**effacement des défauts**, résout généralement le problème. Le FAP retrouve ses performances et le code ne revient pas — à condition d'avoir aussi traité la cause (usage, EGR). Le choix nettoyage/remplacement est détaillé dans **[nettoyage FAP ou remplacement](/blog/nettoyage-fap-ou-remplacement)**.

## Quand le remplacement s'impose

Le remplacement devient nécessaire si la **céramique est détruite** (fondue, fissurée) ou si le filtre est saturé de cendres de façon irréversible. Ces cas existent, mais sont **moins fréquents** qu'on ne le croit. Le diagnostic les identifie clairement, pour éviter de remplacer un filtre qui aurait pu être nettoyé.

## Le rôle du capteur de pression différentielle

Le **capteur de pression différentielle** mesure l'écart de pression de part et d'autre du FAP : c'est lui qui informe le calculateur du colmatage et déclenche les régénérations. Un capteur défaillant peut générer de **faux codes** (P244A/B, P2452/3). Le vérifier évite de remplacer le filtre alors que seul le capteur est en cause.

## Éviter le retour des codes FAP

Pour qu'un code ne revienne pas, il faut traiter la **cause de fond** : favoriser les régénérations (trajets routiers), entretenir l'**EGR** et l'**additif**, et adopter de bons réflexes d'usage. On détaille tout dans **[comment éviter de boucher son FAP](/blog/eviter-fap-bouche)**. Un code effacé sans correction de la cause finit toujours par réapparaître.

## Codes FAP et voyant au tableau de bord

Un code FAP s'accompagne souvent d'un **voyant** au tableau de bord (voyant FAP ou voyant moteur). Le voyant alerte le conducteur, le code précise la nature du problème. Les deux vont de pair : le voyant dit « il y a un souci », le code dit « lequel ». On détaille la marche à suivre face au voyant dans **[voyant FAP allumé : que faire](/blog/voyant-fap-allume-que-faire)**.

## Plusieurs codes en même temps : que comprendre ?

Il n'est pas rare de lire **plusieurs codes** simultanément (FAP, EGR, capteur). Cela ne veut pas dire que tout est à remplacer : souvent, une cause commune (un FAP colmaté, une EGR encrassée) déclenche plusieurs codes liés. Un diagnostic global démêle ces interactions et identifie la **cause racine**, plutôt que de traiter chaque code isolément.

## Le diagnostic, plus important que le code lui-même

Un code défaut est un **point de départ**, pas une conclusion. Deux véhicules avec le même code peuvent nécessiter des interventions différentes selon l'état réel du filtre et des capteurs. C'est pourquoi le **diagnostic** — contre-pression, état de l'additif, test des capteurs — vaut bien plus que la simple lecture du code. Il transforme une information brute en décision juste.

## FAQ

**Le code P2002 veut-il dire que mon FAP est mort ?**
Non. P2002 signale une efficacité sous le seuil, souvent due à un colmatage nettoyable ou à un capteur. Un diagnostic confirme s'il faut nettoyer ou remplacer.

**Peut-on juste effacer un code FAP ?**
On peut, mais s'il y a une cause réelle, le code reviendra. Effacer n'a de sens qu'après avoir traité le problème de fond.

**Que signifie P242F ?**
Une restriction du FAP par accumulation de cendres. Un nettoyage en profondeur évacue une grande partie de ces cendres, ce qu'une régénération ne fait pas.

**Un code FAP impose-t-il un remplacement ?**
Rarement. La plupart des codes traduisent un colmatage ou un capteur. Le remplacement n'est nécessaire que si la céramique est détruite.

**Combien coûte la résolution d'un code FAP ?**
Un nettoyage dès 250 € (décalaminage offert) ; un capteur, plus modeste ; un FAP neuf, plusieurs centaines d'euros. Le diagnostic oriente vers la solution la moins coûteuse adaptée.
